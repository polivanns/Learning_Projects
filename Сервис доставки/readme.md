## О проекте
🎓 Учебный проект

Данный проект выполнен в рамках учебной программы "Бизнес- и системный аналитик" и моделирует реальные практики бизнес-анализа на примере гипотетических стейкхолдеров и условий.

Выполненные задания в папке tasks
## Постановка задачи

Создание  B2B2C-платформы управления службой доставки, которая связывает поставщиков заказов, курьеров, диспетчеров и бухгалтерию. На первом этапе заказы от магазинов и предприятий питания вводятся вручную оператором в единый формат. Система формирует пул свободных заказов, откуда курьеры через мобильное приложение выбирают, бронируют, забирают и доставляют заказы, оперативно обновляя статусы. Диспетчер в веб-интерфейсе отслеживает выполнение и может переназначать заказы. Платформа автоматически обменивается данными с внешней бухгалтерской системой: передаёт заказы для расчётов с поставщиками и подтверждения доставок для начисления оплаты курьерам, а также получает расчётные суммы для отображения в личном кабинете курьера. Администратор управляет регистрацией пользователей и ролями доступа.
## Ценность проекта
Централизованная платформа управления заказами и доставкой (B2B2C), которая сокращает время от получения заказа до доставки, обеспечивает прозрачность статусов доставки, автоматизирует финансовые расчёты и является масштабируемой основой для роста.
